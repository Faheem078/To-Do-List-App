## How to Use

1. Open `index.html` in your web browser
2. Type your task in the input field
3. Click "Add" button or press Enter to add the task
4. Click "Delete" to remove a task
5. Your tasks are automatically saved in localStorage

## Technologies Used

- **HTML5** - Semantic structure
- **CSS3** - Glassmorphism effects, gradient backgrounds, animations
- **Vanilla JavaScript** - Task management and localStorage integration
